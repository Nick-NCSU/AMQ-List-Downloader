# Anki

## Initial Setup

This setup should only need to be completed once. If you have set up the
deck previously, skip to the import deck section.

1. Download the following file: [link](./assets/AMQListDownloader.apkg)

2. In Anki select `File > Import` and select the downloaded file.

3. Select `Import` in the top right.

4. Select the `AMQListDownloader` deck you just imported.

5. Select `Browse`.

6. Right click the card in the deck and select `Notes > Delete`.

7. Modify the deck settings as needed. 
    - Ex: changing the new card gather order to `Random cards`.

## Import Deck

1. In Anki select `File > Import` and select your `./data/anki.txt` file.

2. In import options ensure the correct deck is selected (default `AMQListDownloader`).

3. Select `Import` in the top right.

4. Select `Tools > Check Media`.

5. If you have missing files then select `View Files` and copy the contents of `./data/songs/` into the folder which opened.

6. If you have unused files then you can select `Delete Unused` to delete them from your system.
